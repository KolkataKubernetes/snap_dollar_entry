# Goals
- We now have (somewhat) promising reduced form results when looking at the pipeline built out in /Users/indermajumdar/Research/snap_dollar_entry/1_code/1_2_reduced_form/1_2_0_county/1_2_0_1_regs. We'd now like to pull some descriptives on how food access changes as a function of dollar store entry, and whether that pattern coincides with ABAWD waiver conferral at the county level.
- A few guiding questions for our analysis:
  - How does "food access" vary between treated counties and untreated counties, prior to ABAWD waiver conferral?
  - How does "food access" vary between treated counties and untreated counties, after ABAWD waiver conferral?
  - Do these patterns vary by household income group?
  - What does the distribution of access look like? Does this distribution change after ABAWD waiver conferral? And, does it vary by income group?
  - What does Urban/rural heterogeneity look like?
  - Are there any noticeable differences by retailer format?
- A few definitions for us to work with:
  - We have previously defined retailer format in the pipeline. I want to maintain the same groups when doing this analysis.
  - We will define access by distance to the nearest retailer of interest. We are abstracting away from product assortments for now.
- A challenge with measurement, and initial variable construction:
  - The SNAP retailer panel has lat/longs that should be usable for calculating distances. However, we need to know how to calculate distances FROM census tracts TO each retailer, and how to weight those distances by the population of each census tract.
  - First, we will need to calculate the distance FROM each census tract to the nearest retailer in each retailer format. This will be computationally intensive, so keep in mind we will likely want to use some sort of spatial join or other spatial analysis strategy.
  - Then, once that is saved, we will want to create a set of population weighted distance measures for each county, where weights are constructed by a weighted average of census tract distances (where the weights are by some relevant population). I detail each weighting regime below in ## Metrics.
